SKILL / ADD-REMOTE-INSTALLER

Add Remote Installer

Context-gathering orchestrator that detects your GitHub repo, locates the primary app script, collects install parameters, then delegates to the remote-installer skill to build both artifacts.

1
Auto-detect Repository Context
Silently runs before asking the user anything — parses OWNER/REPO from the git remote URL and checks for conventional launcher scripts.
git remote get-url origin Test-Path scripts/Start-App.ps1 Test-Path .env.example
2
Locate the Primary App Script
The script that actually runs the application (not the launcher). Resolved automatically, or asked if ambiguous.

PATH A — Auto-resolved

→ Read scripts/Start-App.ps1, find the & $target line → extract $APP_SCRIPT

PATH B — AskUserQuestion

→ Launcher not found or target unparseable → present detected script options to user
3
Ask for Required Context
A single AskUserQuestion call. Only asks for what could not be auto-detected — OWNER, REPO, and APP_SCRIPT are never re-asked if resolved.
Install directory PowerShell module (if Import-Module detected)
· Default install dir: C:\osm\<repo-slug>
· Module question skipped if no Import-Module in app script
4
Invoke the remote-installer Skill
Delegates all code generation to the remote-installer skill, passing the full resolved context as structured variables.

VARIABLES PASSED

$OWNER/$REPO
$INSTALL_DIR
$APP_SCRIPT
$START_SCRIPT
$MODULE / $MODULE_VERSION
$ENV_EXAMPLE ($true/$false)

ARTIFACTS PRODUCED

→install.ps1 at repo root
→Update-check block in $APP_SCRIPT
5
Verify Before Finishing
Confirms both artifacts exist and the safeguards checklist passes before reporting success.
Test-Path install.ps1 grep "Update check" $APP_SCRIPT safeguards checklist
FINAL OUTPUT
Two Artifacts Written to Repo
install.ps1
Remote bootstrap at repo root — downloadable as a one-liner. Handles auto-elevation, version check, ZIP download, .env merge, module install.
Update-check block
Inserted into $APP_SCRIPT — silently checks GitHub Releases on every launch and self-updates when a newer version exists.
ONE-LINER TO SHARE
irm 'https://raw.githubusercontent.com/$OWNER/$REPO/main/install.ps1' | iex
