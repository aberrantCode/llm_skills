Logo
Restylizer

Creates styled variants of existing logos and icons, previews with the user, iterates on feedback, and propagates accepted designs across the codebase.

SVG (preferred)
PNG/JPG/WebP
01
Extract Intent from Prompt
Two required signals: source image path + transformation intent. If either is absent, use AskUserQuestion — never ask as plain text. Ask once, combining all missing questions.
AskUserQuestion source image path transformation intent
02
Locate and Read Source File
Use Glob or Read to find the file. If ambiguous, search assets/ recursively. Determine type: SVG (read contents, transform directly) or raster (Python/Pillow script).
Glob Read SVG vs raster decision
03
Apply Transformation
SVG: edit XML directly (fill/stroke replacements, filter elements, gradient stops). Raster: Python + Pillow script. Preserve viewBox, xmlns, all geometry.
Edit SVG XML Pillow for raster Preserve filter IDs
04
Determine Output Path
Default: same dir as source + descriptive suffix (e.g., -dark, -amber-neon). Revision passes: overwrite working file, no new copy per iteration.
homeradar-logo-dark.svg app-icon-restyled.png
05
Write Output + Update Assets Registry
Write file via Write tool. Locate docs/assets.md — add/update row in category table, or create the registry if it doesn't exist. Skip registry on revision passes.
Write tool docs/assets.md table format
06
Preview and Review Loop
Open file with system viewer (cmd.exe start / open / xdg-open). Use AskUserQuestion with 3 choices: Looks good / I'd like changes / Abandon. Repeat until accepted or abandoned.
AskUserQuestion cmd.exe start iterate until satisfied
07
Propagate to All Usages (Optional)
If user accepts: search for all references via Grep, generate each required variant (SVG copies, PNG sizes for manifests, cairosvg for raster exports), update all source file references.
Grep all references manifest.json icons cairosvg for PNG

SVG Path — Preferred

  • Color: replace fill/stroke/stop-color values
  • Neon/glow: add feGaussianBlur + feMerge filter
  • Dark/light: swap background fills, invert foreground
  • Theme: replace entire color palette
  • Preserve viewBox, xmlns, all geometry
  • Update filter="url(#...)" references if renamed

Raster Path — PNG/JPG/WebP

  • Write Python script using Pillow
  • Run via Bash tool
  • If Pillow unavailable: inform user, suggest SVG conversion
  • Propagation: cairosvg for SVG-to-PNG at specific sizes
  • Resize with aspect-ratio preservation
  • Handles manifest icon arrays (16/32/48/128px)

Review Loop Outcomes

Accepted

Looks Good

Proceed to propagation prompt. Update assets registry. End turn if no propagation.

Revision

I'd Like Changes

Return to Step 1. Working file is the new source. Changes accumulate, don't restart.

Abandoned

Delete File

rm output file. Remove entry from docs/assets.md. Respond "Deleted. No changes kept."

Edge Cases

Source not found Use Glob to search; if still absent, AskUserQuestion
Ambiguous transformation AskUserQuestion for visual direction
User wants to overwrite original Confirm with AskUserQuestion first
Pillow unavailable Inform user; suggest install or SVG conversion
cairosvg unavailable Notify for that format; continue with others
Headless/remote environment State file path; continue with review prompt
