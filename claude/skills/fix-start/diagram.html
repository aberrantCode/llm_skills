homeradar-fix-start

Classify severity first — then follow the matching track. SEV-1 requires a Fix Brief and human approval before any worktree is created. All severities commit with spec references.

Step 1 — Classify Severity
SEV-1
Broken / data loss / security regression. System is fundamentally impaired or unsafe.
Critical
SEV-2
Degraded behavior or wrong output. Feature works but produces incorrect results.
Important
SEV-3
Cosmetic, minor, or non-blocking. Does not affect core functionality.
Minor
