CESIUMJS RENDERING NOTES
Custom Data Source
Use Cesium.CustomDataSource per layer — independently removable without affecting other layers.
High-Count Optimization
Use Cesium.PointPrimitive via PointPrimitiveCollection for layers with >1 000 entities to preserve frame rate.
Style Preset Compatibility
Entities must remain visible across all 8 style presets: Normal, CRT, NVG, FLIR, Anime, Noir, Snow, AI.
Entity ID Scheme
Deterministic: <PREFIX>-<index padded to 4 digits> e.g. FLT-0042
PANOPTIC / SPARSE
Entity visibility must respect the global density toggle registered in layerRegistry.js.
Large-Graph Loading
Priority order: Trunk → Secondary → Local. Emit LAYER_LOAD_PROGRESS after each stage for incremental count updates.
