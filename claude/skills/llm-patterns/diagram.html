LLM Patterns

AI-first application patterns, LLM testing strategies, and prompt management for production systems.

LLM for logic, code for plumbing.
Division of Responsibility
Use LLM for
Semantic & Cognitive Work
  • Classification, extraction, summarization
  • Decision-making with natural language reasoning
  • Content generation & transformation
  • Complex conditional logic (brittle in code)
Use Code for
Deterministic Plumbing
  • Data validation (Zod / Pydantic)
  • API routing & HTTP handling
  • Database operations
  • Auth/authorization, orchestration, error handling

Prompt Management
01 — TEMPLATE
Template Functions
Prompts as typed functions with interpolated context. One file per prompt domain.
02 — VERSION
Version Registry
Export named versions (v1, v2) with a current pointer. Track accuracy over time.
03 — SCHEMA
Zod Schemas
All LLM responses validated via Zod. Never trust unvalidated JSON from an LLM.
Three-Tier LLM Testing Strategy
Tier 1 — Unit Tests (Mocks)
Mock the LLM client. Fast, deterministic. Tests that your service logic handles the response correctly. Run on every commit.
every CI run
Tier 2 — Fixture Tests
Saved real LLM responses as JSON fixtures. Tests parsing & schema validation without calling the API. Still deterministic.
every CI run
Tier 3 — Evaluation Tests
Live API calls against labeled test cases. Measures accuracy >90%. Slow & expensive — run nightly or on-demand via RUN_LLM_EVALS=true.
nightly · on-demand
Cost & Performance Tracking
Track every LLM call with: model · inputTokens · outputTokens · latencyMs · cost. Log or pipe to your observability stack. LLM calls are slow by design — build async-first and never block UI.
Anti-Patterns to Avoid
  • Hardcoded prompts in business logic
  • No schema validation on LLM responses
  • Live LLM calls in CI unit tests
  • No cost tracking
  • Ignoring latency — LLM calls are slow
  • No fallback for timeouts & failures
  • Prompts without version control
  • No evaluation suite — measure accuracy
  • LLM for deterministic logic (auth, math)
  • Giant monolithic prompts
