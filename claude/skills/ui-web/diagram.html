Glassmorphism — When to Use

Use For

  • Hero sections with image backgrounds
  • Floating cards over gradients
  • Modal overlays
  • Navigation bars (subtle)
  • Feature highlights

Avoid For

  • Every card (overuse kills effect)
  • Text-heavy content areas
  • Forms (reduces contrast)
  • Data tables
