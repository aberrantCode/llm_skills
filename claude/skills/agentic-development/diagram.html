Skill Reference · agentic-development

Agentic Development

Build autonomous AI agents using Pydantic AI (Python) or the Claude Agent SDK (Node.js). Agents reason, plan, use tools, and verify their own work.

Core Principle: Plan first, act incrementally, verify always. Agents that research and plan before executing consistently outperform those that jump straight to action.
Framework Selection by Language
Python Default
Pydantic AI
Type-safe structured outputs via Pydantic models. Multi-model support, production-ready. Use result_type=YourModel for validated responses.
Node.js / Next.js Default
Claude Agent SDK
Official Anthropic SDK. Native tools, multi-agent support, streaming. Implements the agentic loop with stop_reason === "tool_use".
Agent Architecture — Three Components
◈
Model (Brain)
LLM for reasoning & decision-making. Match complexity to task.
⊞
Tools (Arms/Legs)
APIs, functions, external systems. Defined with schemas & risk levels.
≡
Instructions (Rules)
System prompts defining behavior & scope boundaries.

Guardrails & Safety — Multi-Layer
Input Validation
  • Input safety classifier
  • Scope boundary check
  • Intent verification
Tool Risk Levels
  • low read operations
  • medium write with confirm
  • high human-in-the-loop
Output Validation
  • Output safety check
  • Schema validation
  • Scope compliance
Memory & State Management
Short-term
Conversation Memory
Messages array with token budget. Summarize when approaching limit. Maintain current plan & completed steps across tool calls.
Long-term
Persistent Memory
Store learnings across sessions. Semantic search over past interactions. Structured key/value with vector embeddings for similarity.

Agent Development Checklist
  • Define clear agent scope & boundaries
  • Design tools with explicit schemas & risk levels
  • Implement explore–plan–execute–verify workflow
  • Add multi-layer guardrails
  • Set up conversation & persistent memory
  • Write behavior & evaluation tests
  • Configure appropriate model for task complexity
  • Add human-in-the-loop for high-risk operations
  • Monitor token usage & costs
  • Document skills & instructions
