iOS — Human Interface Guidelines

  • Typography SF Pro / System
  • Large Title 34pt bold
  • Body text 17pt regular
  • Corner radius 10–14pt
  • Default tint Blue #007AFF
  • Shadows shadowOpacity 0.08–0.15
  • Home indicator paddingBottom: 34
  • Tab bar height 83px (with home)

Android — Material Design 3

  • Typography Roboto
  • Medium Title 22pt
  • Body text 16pt regular
  • Corner radius 12–28pt
  • Dynamic color Material You
  • Shadows elevation: 2–8
  • No home indicator paddingBottom: 10
  • Ripple feedback android_ripple prop
Spacing

4px Base Grid

  • xs: 4 · sm: 8
  • md: 16 · lg: 24
  • xl: 32 · 2xl: 48
  • useSafeAreaInsets()
  • Math.max(insets.left, 16)
Animations

Native Driver First

  • useNativeDriver: true
  • 200–300ms duration
  • Animated.spring() for physics
  • useSharedValue() (Reanimated)
  • withSpring() on press
Loading States

Skeleton + Indicator

  • withRepeat + withSequence
  • Opacity 0.3 → 1 pulse
  • ActivityIndicator native
  • disabled={loading} on button
  • Platform-specific sizing

Premium Feel Checklist

All touch targets 44pt+
Consistent 4pt grid spacing
Platform-appropriate styling
Safe area handling (useSafeAreaInsets)
Native animations at 60fps
Skeleton loaders for loading states
Dark mode color scheme support
Accessibility labels on all elements
Haptic feedback on key actions
Pull-to-refresh where appropriate
