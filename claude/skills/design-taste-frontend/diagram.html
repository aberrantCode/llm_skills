Rule 01

Deterministic Typography

  • Display: text-4xl md:text-6xl tracking-tighter
  • Geist, Outfit, Cabinet Grotesk, Satoshi
  • Inter font: BANNED
  • Serif banned on Dashboard/Software UI
  • Body: text-base text-gray-600 max-w-[65ch]
Rule 02

Color Calibration

  • Max 1 accent color, saturation <80%
  • AI Purple/Blue aesthetic: BANNED
  • No purple button glows, no neon gradients
  • Neutral base: Zinc/Slate
  • Accents: Emerald, Electric Blue, Deep Rose
Rule 03

Layout Diversification

  • Centered Hero: BANNED when Variance >4
  • Force Split Screen (50/50)
  • Left-content / Right-asset layouts
  • Asymmetric whitespace structures
  • Mobile: collapse to single-column
Rule 04

Materiality & Anti-Card

  • Dense UI: generic cards BANNED
  • Use border-t, divide-y, negative space
  • Cards only when elevation signals hierarchy
  • Shadow tinted to background hue
  • Rounded-[2.5rem] for major containers
Rule 05

Interactive States

  • Loading: skeletal loaders (no generic spinners)
  • Empty states: beautifully composed
  • Error: clear inline reporting
  • :active → -translate-y-[1px] or scale-[0.98]
  • All three states always implemented
Rule 06

Data & Form Patterns

  • Label above input — always
  • Helper text in markup
  • Error text below input
  • Standard gap-2 for input blocks
  • Zod validation on all inputs
