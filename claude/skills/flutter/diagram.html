Architecture

Three-Layer Project Structure

Presentation Screens, widgets, Riverpod providers, ConsumerWidget lib/presentation/
Domain Business entities, repository interfaces (no implementation) lib/domain/
Data Freezed models, repository implementations, API services lib/data/

AsyncValue Handling

Three States of Every Async Provider

⏳
AsyncLoading()
Show spinner or skeleton. Provider is fetching.
✓
AsyncData(value)
Render content. Data is available.
✕
AsyncError(error)
Show error UI with retry button.
usersAsync.when() Handles all three states with callbacks — preferred pattern
switch (usersAsync) Pattern matching alternative using Dart 3 sealed classes
