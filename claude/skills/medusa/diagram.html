Extension Points
Backend Logic
Modules
Custom business logic as services. Registered in medusa-config.ts. Resolved via IoC container in API routes.
Multi-step Process
Workflows
Steps with compensation (rollback) functions. Transactional, retry-safe. Created via createWorkflow().
Event Listeners
Subscribers
React to system events (order.placed, product.created). Defined via SubscriberConfig.event.
Cron Tasks
Scheduled Jobs
Background jobs with cron schedule. Access full DI container. Located in src/jobs/.
Dashboard
Admin UI Extensions
Widgets (inject into existing pages) and custom routes (new admin pages). Uses @medusajs/ui components.
HTTP Layer
Custom API Routes
File-based routing in src/api/store/ or src/api/admin/. Export named HTTP methods (GET, POST, PUT, DELETE).

Setup Checklist
Setup
  • PostgreSQL configured
  • Redis configured (recommended)
  • Admin user created via CLI
  • CORS origins configured
  • JWT/Cookie secrets set
Anti-Patterns
  • Never edit .medusa/ folder
  • Use services, not direct DB access
  • Use workflows for complex operations
  • No hardcoded URLs — use env vars
  • Don't ignore TypeScript errors
