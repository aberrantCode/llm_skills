Kotlin Anti-Patterns — Avoid

runBlocking on main thread
GlobalScope (use structured concurrency)
Collecting flows in init{}
Exposing MutableStateFlow publicly
No catch {} in flow pipelines
lateinit for nullable values
Hardcoded Dispatchers (inject them)
Side effects in Composables
Unstable Compose parameters
Skipping sealed classes for state
