Android Anti-Patterns — Avoid These

  • Context leaks in static fields
  • Memory leaks in callbacks
  • UI updates on background thread
  • Hardcoded strings (use resources)
  • God Activities (>200 lines)
  • NetworkOnMainThreadException
  • Ignoring lifecycle states
  • Blocking the main thread (>16ms)
  • Not handling config changes
  • Hardcoded dimensions (use dp/sp)
  • Deep view hierarchies (>10 levels)
  • Not closing Cursors/streams
