Checklists
Integration
  • Profile sync on signup/update
  • Key events tracked (view, cart, order)
  • Order events include Items array
  • Revenue tracked with $value
  • Unique IDs for deduplication
Testing
  • Test profile creation
  • Verify events in dashboard
  • Test webhook delivery
  • Test rate limit handling
Anti-Patterns
  • Missing email or phone identifier
  • Duplicate events — use unique_id
  • Missing Items array on orders
  • Client-side only tracking
  • Missing revenue ($value)
