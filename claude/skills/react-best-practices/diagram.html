2. Bundle Size Optimization CRITICAL 5 rules
bundle-barrel-imports Import directly from source, not barrel index.ts files. Barrel imports prevent tree-shaking by forcing the entire module to load.
bundle-dynamic-imports Use next/dynamic for heavy components. Lazy load routes, modals, charts, and third-party widgets that aren't needed on initial render.
