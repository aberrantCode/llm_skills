List Performance — CRITICAL

8 Rules for Fast Lists

list-performance-virtualize
Use FlashList instead of FlatList for large datasets. Recycles components instead of remounting.
list-performance-item-memo
Wrap list item components with React.memo(). Prevents re-render on every parent update.
list-performance-callbacks
Stabilize callbacks with useCallback. Unstable refs cause memo to miss and re-render.
list-performance-inline-objects
No inline style objects in list items. New object reference on every render breaks memo.
list-performance-images
Optimize images in lists. Use expo-image with proper sizing and cache policy.
list-performance-item-types
Use item types for heterogeneous lists. Allows FlashList to recycle correctly.
Animation — HIGH

Reanimated Best Practices

animation-gpu-properties
Animate only transform and opacity. Other props trigger JS-thread layout recalculation.
animation-derived-value
Use useDerivedValue for computed animation values. Keeps calculations on the UI thread.
animation-gesture-detector-press
Use Gesture.Tap() from gesture-handler instead of Pressable for gesture-driven interactions.
UI Patterns — HIGH

Native UI Rules

ui-expo-image
Use expo-image for all images. Blurhash placeholder, better caching, prefetching.
ui-pressable
Use Pressable over TouchableOpacity. More flexible, better feedback, future-proof.
ui-safe-area-scroll
Handle safe areas in ScrollViews. Use contentInsetAdjustmentBehavior or insets.
ui-native-modals
Use native modals when possible. Native sheet behaviors, spring animations, gestures.
ui-styling
Use StyleSheet.create or Nativewind. Never inline style objects in render.
navigation-native-navigators
Use native stack and native tabs over JS navigators for 60fps transitions.
State + Monorepo + Config — MEDIUM to LOW

Remaining Rules

react-state-minimize
Minimize state subscriptions. Subscribe to derived booleans, not raw values where possible.
react-state-dispatcher
Use dispatcher pattern for callbacks to avoid re-render propagation through callbacks.
rendering-text-in-text-component
Always wrap text in <Text> components. Raw strings outside Text cause crashes.
rendering-no-falsy-and
Avoid falsy && for conditional rendering. Use ternary or explicit null checks.
monorepo-native-deps-in-app
Keep native dependencies in the app package, not shared packages in monorepos.
fonts-config-plugin
Use config plugins for custom fonts. Avoids manual native linking issues.
