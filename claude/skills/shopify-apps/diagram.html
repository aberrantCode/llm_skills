Skill Reference · shopify-apps

Shopify App Development

Build Shopify apps with Remix, the Admin GraphQL API, and checkout UI extensions. Stack: Remix + TypeScript + Polaris + Prisma.

Prerequisites & Quick Start
01 — ACCOUNT
Partner Account
Sign up at partners.shopify.com (free). Create a development store.
02 — CLI
Shopify CLI
npm i -g @shopify/cli · Node 18.20+
03 — SCAFFOLD
Init Remix App
shopify app init → Remix template
04 — DEV
Start Dev Server
shopify app dev (tunneled)

Checklists
Before Development
  • Partner account created
  • Development store created
  • App created in Partner Dashboard
  • Shopify CLI installed
  • App scaffolded with Remix template
Before App Store Submission
  • GDPR webhooks implemented
  • App uninstall webhook cleans data
  • No hardcoded API keys
  • Rate limit handling with exponential backoff
  • Session tokens validated (HMAC via SDK)
  • Privacy policy URL configured
  • Use GraphQL only (REST deprecated)
