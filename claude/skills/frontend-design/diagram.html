Aesthetic Directions — Pick One, Execute Fully

Brutally Minimal

Nothing decorative. Pure function. Extreme whitespace, one typeface, no color.

Maximalist Chaos

Dense layers, clashing type, multiple palettes. Deliberately overwhelming.

Organic / Natural

Warm earthy tones, organic shapes, imperfect textures and paper-like surfaces.

Luxury / Refined

Deep navy + gold, generous negative space, serif type. Whispers don't shout.

Brutalist / Raw

Stark borders, visible structure, monospace everywhere, unpolished honesty.

Art Deco / Geometric

Angular symmetry, geometric ornament, strong verticals, metallic accents.

Industrial / Utilitarian

Functional type, grid rigidity, data-forward, no decoration without purpose.

Editorial / Magazine

Mixed type scales, pull quotes, asymmetric columns, editorial photography.

Typography

  • Beautiful & distinctive fonts
  • Avoid Arial, Inter, Roboto
  • Display + body pairing
  • Unexpected + characterful

Color

  • Cohesive — CSS variables
  • Dominant + sharp accents
  • Avoid purple/pink/cyan
  • Vary light vs dark themes

Motion

  • Staggered page load reveals
  • CSS-first for HTML artifacts
  • Motion library for React
  • Surprise on hover + scroll

Composition

  • Unexpected layouts
  • Asymmetry + overlap
  • Diagonal flow
  • Generous negative space

Background

  • Atmosphere + depth
  • Gradient meshes, noise
  • Geometric patterns
  • Dramatic shadows

Never Converge on These Generic Patterns

Inter / Roboto / Arial as primary font
Purple gradients on white backgrounds
Space Grotesk across all generations
Predictable 3-card feature rows
Default Tailwind purple/pink/cyan palette
Centered hero over dark background image
Cookie-cutter layouts with no context
Same aesthetic every generation
